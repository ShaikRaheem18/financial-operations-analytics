# Financial Operations Analytics & Dashboard

## Project Overview
This project simulates a real-world financial operations analytics system. 
Raw credit card transaction data is cleaned using Python, structured into a relational MySQL database, analyzed using SQL, and visualized through Power BI dashboards.

## Dataset
Source: Credit Card Transactions Dataset (Kaggle)

## Tools Used
- Python (pandas, numpy)
- MySQL
- SQL
- Power BI
- Git & GitHub
